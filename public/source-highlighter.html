The bookmarklet looks for elements with:

  • id="section-name" - Standard HTML ID attributes
  • name="section-name" - Anchor name attributes

When a matching element is found, it will:

  • Scroll to the element smoothly
  • Apply a yellow highlight with border
  • Add a subtle shadow effect
  • Remove the highlighting after 5 seconds
